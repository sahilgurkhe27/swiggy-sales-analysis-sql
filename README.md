# 🍽️ Swiggy Sales Data Analysis (SQL)

This project focuses on analyzing Swiggy food delivery data using SQL to understand customer behavior, sales trends, and restaurant performance. The goal is to turn raw data into useful insights that can help in better business decisions.

---

## 🔧 Tools Used
- SQL (SQL Server)
- Excel / CSV Dataset

---

## 📌 Project Overview
In this project, I worked on cleaning and analyzing the dataset to make it ready for analysis. Then, I used SQL queries to explore key business metrics and trends.

Main tasks performed:
- Data validation (checking null values and empty fields)
- Removing duplicate records
- Preparing clean data for analysis

---

## 📊 Key KPIs
- Total number of orders  
- Total revenue generated  
- Average dish price  
- Average customer rating  

---

## 🔍 Business Insights
- Identified top cities with highest number of orders  
- Found restaurants generating maximum revenue  
- Analyzed highest rated restaurants  
- Discovered most ordered dishes  
- Studied monthly order trends  

---

## 💡 What I Learned
- Writing SQL queries for real-world data analysis  
- Data cleaning and preprocessing techniques  
- Using aggregate functions and grouping  
- Extracting meaningful insights from raw data  

---

## 🚀 Conclusion
This project shows how SQL can be used to analyze food delivery data and generate insights about customer preferences, popular dishes, and business performance.
